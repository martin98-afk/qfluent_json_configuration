# Excel文件结构说明

## 标准工况Excel格式

### Sheet结构

每个Sheet代表一种控制参数：

| 区域 | 说明 |
|------|------|
| Sheet名 | 控制参数名称（如：引风机频率、脱硫消石灰下料量等） |
| 第1列 | 控制参数占位列（用于标记工况分组） |
| 第2列~N列 | 工况参数列（从第二列开始直到"下控范围"列之前） |
| "下控范围"列附近 | 包含"寻优上限"和"寻优下限"列 |
| 后续列 | 该工况下原始数据分布等辅助信息 |

### 通用列结构

```
| 控制参数占位 | 工况参数1 | 工况参数2 | ... | 工况参数N | 寻优上限 | 寻优下限 | 原始数据... |
```

### 关键特点

1. **第一列**：控制参数占位标识，不参与计算
2. **工况参数**：从第二列开始，是计算寻优范围的依据
3. **寻优列**：通常成对出现，列名可能包含"寻优"、"上限"、"下限"等关键词
4. **Sheet名**：标识当前控制参数类型

## 脚本编写要点

### 动态识别列

```python
# 查找"寻优上限"和"寻优下限"列
for col_idx, header in enumerate(headers):
    if header and "上限" in str(header):
        upper_col_idx = col_idx + 1  # 转为1-based
    elif header and "下限" in str(header):
        lower_col_idx = col_idx + 1
```

### 工况参数范围

工况参数列索引范围为 `[1, lower_col_idx - 1)`（0-based），即：
- 起始：第2列（索引1）
- 结束：寻优下限列之前

### 批处理模式

```bash
# 单文件处理
python script.py input.xlsx output.xlsx

# 批量处理
# 无参数时自动扫描当前目录的 output*.xlsx 文件
python script.py
```
